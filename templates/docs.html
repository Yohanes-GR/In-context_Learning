How To Use The API

1. api/jdentities
Request: 
    
    http://[ip_address]/api/jdentities
    json body
        {
            "j_d":"Your Job Description",
            "ch_model":1 or 2  // 1-> Default Model('xlarge'). 
                            // 2-> Fine Tuned Model
            "pr_type": 1 or 2   // 1-> Default or Pick 3 Similar Job Description from Trian Data. 
                                // 2-> Pick 3 Random Job Description from Trian Data.
            "num_token": 100-250 // number of tokens to generate
            "num_example": 1-5 // number of Examples to use
        }
Response: 
    
    http://[ip_address]/api/jdentities
